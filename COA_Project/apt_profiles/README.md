# APT profiles (open-source summaries)

These YAML files are **heuristic templates for education and demo matching** inside C.O.A. They condense publicly documented TTPs and sectors (e.g. [MITRE ATT&CK Groups](https://attack.mitre.org/groups/)) and **must not** be treated as classified intelligence.

Expand or replace profiles using only sources you are allowed to cite in your environment.
